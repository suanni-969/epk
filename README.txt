# SUANNI EPK - Setup Instructions

## To view the EPK with photos:

### Option 1: View locally (RECOMMENDED)
1. Download the entire EPK package (HTML file + photos folder)
2. Keep the folder structure intact:
   ```
   suanni_epk/
   ├── suanni_epk.html
   └── photos/
       ├── 1.png
       ├── 2.jpg
       ├── 3.jpg
       └── ... (all 11 photos)
   ```
3. Open `suanni_epk.html` in your browser

### Option 2: Host online
Upload both the HTML file and the photos folder to your web hosting service, maintaining the same folder structure.
